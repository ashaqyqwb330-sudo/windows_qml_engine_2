# 🌐 محرك الوصول الموحد للويندوز بلغة بايثون و QML (المحرك النقي)

أهلاً بك يا صديقي في النسخة المكتبية فائقة الأداء من **المنصة الذهبية الذكية (Golden Platform Engine)** المخصصة لنظام تشغيل ويندوز (Windows). لقد تم تصميم هذا المحرك ليكون نظيفاً، ناصعاً، وقوياً للغاية لمحاكاة وتوسيع نفس القدرات الجبارة للمنصة على حاسوبك الشخصي.

---

## ✨ المميزات والخصائص الأساسية

1. **⚙️ معالج ومستخرج حزم البناء (Pack Extractor)**:
   - تحليل الكتل النصية وتفسير توجيهات `@builder:file <path>` و `@builder:end` لإنشاء واستخراج الملفات البرمجية وهياكل المجلدات تلقائياً في مجلد العمل النشط.
   - تشغيل وتنفيذ توجيهات الأوامر الآمنة `@executor: <command>` مباشرة في الخلفية وعرض مخرجاتها بدقة متناهية.

2. **📦 صانع حزم التصدير الذاتي (Pack Builder)**:
   - زحف وتجميع كامل لمجلد برمجي محلي على جهازك (مع تخطي وتجاهل المجلدات المؤقتة والملفات الثنائية بشكل ذكي).
   - تحويل المجلد البرمجي بأكمله إلى كود حزمة موحد وتغليفه وحفظه ونسخه مباشرة إلى الحافظة (Clipboard) لتكون جاهزة للمشاركة الفورية.

3. **🧠⚡ الالتقاط والتحليل الذكي (Smart Capture & Inbox)**:
   - الصق أي نص في المحرك، وسيقوم النظام فوراً بتحديد نوعه (حزمة بناء، مستند Markdown، نص برمجى أو مذكرة عادية).
   - ميزة **التجميل والتنسيق (Beautifier)**: يقوم تلقائياً بتحويل نصوص Markdown المكتوبة وتنسيقها داخل تصميم HTML غاية في الجمال مدمج به مظهر "الجرانيت الداكن والذهبي" الأسطوري للمنصة.

4. **🎨 واجهة مستخدم أسطورية (Premium QML UI)**:
   - تصميم أنيق يعكس الهوية البصرية الفاخرة لتطبيقنا: خلفية داكنة فاخرة، وتفاصيل وحواف مذهبة متوهجة، وانتقالات وتأثيرات حركية فائقة السلاسة.
   - دعم كامل للشاشات عالية الدقة (High DPI Scaling) لتظهر الواجهة بوضوح كريستالي فائق النقاء.

---

## 🛠️ متطلبات التشغيل والتثبيت السريع

لتشغيل المحرك على نظام ويندوز، يرجى اتباع الخطوات البسيطة التالية:

### 1. تثبيت لغة بايثون (Python)
تأكد من تثبيت بايثون (إصدار 3.8 أو أحدث) على جهازك، وتأكد من تفعيل خيار **"Add Python to PATH"** أثناء التثبيت.

### 2. تثبيت المكتبات المطلوبة (PySide6)
افتح واجهة الأوامر (Command Prompt / PowerShell) في مجلد المشروع، ونفذ الأمر التالي:
```bash
pip install -r requirements.txt
```
أو التثبيت المباشر للمكتبة الرسمية لـ Qt6:
```bash
pip install PySide6
```

### 3. تشغيل المحرك
قم بتشغيل المحرك بنقرة مزدوجة على ملف `main.py` أو عبر واجهة الأوامر:
```bash
python main.py
```

---

## 📁 هيكلية المجلد البرمجي للمحرك النقي

- `main.py`: نقطة انطلاق المحرك المكتبي وتجهيز بيئة عمل Qt6 وتكاملها مع بايثون.
- `engine.py`: العقل المدبر للمنصة والمحرك البرمجي المسؤول عن معالجة النصوص، استخراج الملفات، تجميع المجلدات، وتحليل المحتويات وحفظها.
- `main.qml`: واجهة المستخدم الرسومية الخارقة المصممة بأحدث تقنيات تصميم QML.
- `requirements.txt`: ملف تعريف الاعتمادات والمكتبات البرمجية المطلوبة للتشغيل.
